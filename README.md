# Discord Bot Builder

The name is pretty self-explanatory, it's a powerful program that makes building Discord bots simple. No coding experience is necessary!
# Reporting Bugs

Hey! You wanna report a bug? Great!

You can report bugs on the [Issues](https://github.com/TheDogSys/Discord-Bot-builder/issues) tab.
To help with fixing bugs a good bug report is needed so here's a template to help make good bug report,

    Summary: A short description of the bug

    Operating system: All the operating systems where you found the bug

    Description: A detailed description of the bug

    Expected and Actual Result: What should've happened and what happened?

    Steps to Reproduce: A detailed step by step description of how to perform the bug

## Setting up workspace (Visual Studio Code)

To work with Discord Bot Builder, [Node v16.6.0](https://nodejs.dev/download) or higher is required.

You will also want to download the requried packages using `npm install` in a terminal. Once done you can start the program by running the `npm start` command in a terminal.

*Note: I suggest using having `npm start` as a command for debugging so it's easier to start the program.*
